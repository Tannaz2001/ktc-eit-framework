# 🚀 Quick Start Card

**Just 3 commands to run the dashboard!**

---

## Copy & Paste These Commands

### Command 1️⃣ (First time only)
```powershell
Copy-Item .env.example .env
```

### Command 2️⃣ (Every time you want to start)
```powershell
docker compose up -d
```

### Command 3️⃣ (Open in browser)
```
http://localhost:8501
```

---

## That's It! ✅

You now have a running dashboard!

---

## When You Want to Stop
```powershell
docker compose down
```

---

## Stuck? Try This
```powershell
docker compose restart
```
