#   - tensorflow / opencv-python -> CompetitionCNN (ABC1 CNN post-processor)

# CUQI1, CUQI2_main and ml_inverse_method_2 need nothing beyond requirements.txt
# (pure numpy/scipy/scikit-image).
#
# NOT included: ktc2023_abc2 (torchmetrics), ktc2023_e2e (deepinv),

# leaderboard scores yet. Add torch_geometric/deepinv/torchmetrics
# yourself (all have CPU wheels) if you want them too.
